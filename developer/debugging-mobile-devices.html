    Débogage sur les appareils mobiles

    Dans cette rubrique, vous apprendrez à déboguer JavaScript sur les appareils mobiles. Pendant le développement, les tests et le débogage du contenu sur les plates-formes mobiles peuvent être très difficiles.

    Présentation

    Dans ce document, vous apprendrez comment démarrer le débogage sur les appareils mobiles et couvre les solutions suivantes :

    • Débogage sur les appareils iOS
    • Débogage sur les appareils Android
    • Utilisation de Charles sur les appareils mobiles
    Ce document sert de point de départ pour les développeurs de déboguer sur les appareils mobiles. Il ne couvre pas toutes les méthodes de débogage mobile. Pour plus d'informations sur le débogage en général, consultez la documentation JavaScript Debogging Basics .

    Débogage iOS

    Configuration minimale requise pour le débogage iOS

    • Safari version 6+ sous Mac macOS
    • iPhone ou iPad avec iOS 6+
    Remarque : si vous êtes sous iOS 10.3 ou version ultérieure, ouvrez les paramètres et accédez à Général > À propos > Paramètres de confiance du certificat, recherchez le certificat Charles Proxy et activez le certificat pour activer la confiance totale pour celui-ci.

    Utilisation du débogage à distance Safari

    La mise à jour iOS 6 d'Apple a introduit Safari Remote Debogging, qui vous permet de déboguer des pages Web dans l'application Safari sur les appareils iOS. Pour commencer, suivez les étapes ci-dessous :

    1. Connectez votre appareil iOS à votre machine via un câble USB.
    2. Sur votre appareil, ouvrez l'application Paramètres . Sélectionnez Safari, faites défiler jusqu'au bas de la page et ouvrez le menu Avancé. Activez l'option Inspecteur Web .
    3. Toujours sur votre appareil, ouvrez l'application Safari et accédez à n'importe quelle page Web.
    4. Sur votre Mac, lancez Safari, ouvrez les Préférences (> PréférencesSafari) et accédez à la section Avancé. Activez le menu Afficher le développement dans l'option de la barre de menus .
    5. Ouvrez le menu Développer . Vous verrez votre appareil iOS dans la liste. Lorsque vous passez la souris sur le nom de votre appareil, l'URL du site que vous avez consultée s'affiche sur votre appareil.
    6. Sélectionnez l'URL et une fenêtre Web Inspector s'ouvre, vous permettant de prévisualiser, d'inspecter et de déboguer le contenu de votre navigateur mobile à partir de votre bureau.
    7. Dans la section Ressources , vous pouvez prévisualiser et inspecter la page Web. Le survol des éléments dans l' Inspecteur Web met en surbrillance les éléments correspondants dans le navigateur de votre appareil iOS.
    8. Vous pouvez également utiliser les sections Débogueur et Console pour déboguer davantage votre page en définissant des points d'arrêt, en journalisant des données, etc.
    Pour plus d'informations sur l'utilisation de Safari Web Inspector, consultez la documentation développeur d'Apple.

    Débogage Android

    Configuration minimale requise pour le débogage Android

    • Chrome 32+ s'exécutant sur Mac ou Windows
    • Appareil Android avec Chrome pour Android 32+

    Remarque : Les étapes de cette section ont été écrites à l'aide d'un appareil exécutant Android 4.3.

    Activation du débogage sur votre appareil

    Avant de pouvoir commencer à déboguer du contenu sur votre appareil, vous devez activer le débogage USB dans les paramètres de votre appareil.

    1. Si vous exécutez Android 4.2+, vous devez activer les options développeur de votre appareil avant de pouvoir activer le débogage USB. Sur votre appareil, ouvrez l'application Paramètres et sélectionnez le menu À propos du téléphone . Localisez le numéro de construction de votre appareil et touchez-le sept fois.
      Si vous exécutez une ancienne version d'Android, vous pouvez activer le débogage USB en accédant directement à Paramètres > Options du développeur.
    2. Appuyez sur le bouton Précédent de votre appareil et vous verrez le menu Options du développeur répertorié dans les Paramètres.
    3. Ouvrez le menu Options du développeur et cochez la case pour activer le débogage USB.

    Débogage à distance à l'aide de Chrome pour Android

    Maintenant que le débogage USB est activé sur votre appareil, procédez comme suit pour démarrer le débogage à distance avec Chrome :

    1. Sur votre ordinateur, lancez Chrome et accédez à about:inspect. Sur la page Périphériques , activez l'option Découvrir les périphériques USB .
    2. Connectez votre appareil à votre machine via un câble USB. Sur votre appareil, une boîte de dialogue vous demandera si vous souhaitez autoriser le débogage USB. Choisissez OK pour permettre à votre ordinateur de se connecter à votre appareil.
      Si vous êtes sur un ordinateur Windows, vous devrez peut-être installer des pilotes USB. Pour plus d'informations, consultez la documentation du développeur Android.
    3. Sur votre appareil, ouvrez l'application Chrome et accédez à une page Web.
    4. Revenir à Chrome sur votre ordinateur. Sur la about:inspect page, vous devriez voir votre appareil connecté. Sous le nom de votre appareil, vous pourrez voir une liste de toutes les pages ouvertes dans Chrome sur votre appareil.
    5. Pour commencer le débogage d'une page Web, sélectionnez le lien Inspecter sous l'URL de la page. Cela ouvrira les outils de développementde Chrome.
    6. À gauche de l'onglet Eléments , vous pouvez sélectionner l'icône du périphérique pour ouvrir la fonction Screencast de DevTool. Cela vous permet d'ouvrir un aperçu en direct du contenu tel qu'il apparaît sur votre appareil mobile. Pour plus d'informations sur la façon dont vous pouvez utiliser le screencasting pour interagir avec votre appareil, consultez la documentation de Google.
    Pour plus d'informations sur l'utilisation des outils de développement Chrome, consultez la documentation Google pour les développeurs.

    Charles pour mobile

    Charles est un proxy web qui peut être exécuté sur votre propre ordinateur, vous permettant d'enregistrer les données envoyées et reçues via votre navigateur Web. Visitez le site Web de Charles pour plus d'informations sur cette application.

    Exigences minimales pour Charles

    • Charles est compatible avec Windows, Mac et Linux
    • Appareil iOS et/ou Android

    Configuration de Charles sur votre machine

    Suivez les étapes ci-dessous pour faire fonctionner Charles sur votre machine.

    1. Téléchargez et installez Charles sur votre ordinateur.
    2. Sur votre ordinateur, exécutez Charles. Dans le menu, sélectionnez Paramètres du > proxy proxy. Assurez-vous qu'un port a été attribué (p. ex. 8888) et que l'option Activer le proxy HTTP transparent est sélectionnée.
    3. Sélectionnez OK pour enregistrer les modifications et fermer la boîte de dialogue.
    4. Dans le menu, sélectionnez Proxy > SSL Proxy Settings. Assurez-vous que l'option Activer le proxy SSL est sélectionnée.
    5. Dans la section Emplacement , sélectionnez le bouton Ajouter . Dans la boîte de dialogue Modifier l'emplacement , accédez au champ Port et entrez 443. Sélectionnez OK pour enregistrer cette modification.
    6. Sélectionnez OK pour fermer la boîte de dialogue Modifier l'emplacement .
    7. Sélectionnez OK pour quitter la boîte de dialogue Paramètres de proxy SSL .
    8. Accédez à Aide > SSL Proxying > Installer le certificat racine Charles sur un appareil mobile ou un navigateur distant.
    9. Vous devriez voir un message qui identifie l'adresse IP et le numéro de port du serveur local (8888) que vous utiliserez pour configurer le périphérique. Vous devriez également voir l'URL pour télécharger et installer le certificat. Copiez ces fichiers sur votre ordinateur pour une utilisation ultérieure.
      Notez que vous devrez activer la confiance du certificat Charles.

    Configuration de Charles sur votre appareil

    Dans cette section, vous apprendrez comment configurer votre appareil iOS et/ou Android pour qu'il fonctionne avec Charles sur votre ordinateur.

    Configuration de votre appareil iOS

    1. Sur votre appareil, assurez-vous que vous êtes connecté au même réseau que votre ordinateur. Vous pouvez le vérifier en accédant aux paramètres > Wifi et en sélectionnant le nom du réseau pour afficher les options avancées.
    2. Dans la section HTTP en bas, sélectionnez Mode manuel . Ensuite, ajoutez l'adresse IP du serveur et le numéro de port (8888) que vous avez copiés à partir de Charles lors d'une étape précédente.
    3. Sur votre appareil, ouvrez un navigateur et accédez à http://www.charlesproxy.com/getssl. Vous serez redirigé vers les paramètres de votre appareil où vous serez invité à installer le profil de certificat racine personnalisé Charles Proxy . Cliquez sur le bouton Installer pour installer le certificat Charles sur votre appareil.
    4. Suivez les étapes pour installer le certificat Charles. Lorsque vous avez terminé, sélectionnez Terminé.
    5. Ensuite, vous êtes prêt à vous assurer que votre appareil approuve le certificat Charles. Sur votre appareil, accédez à Généralités à > propos des paramètres > d'approbation des certificats. Ici vous devriez trouver le certificat Charles Proxy.
    6. Activez le certificat Charles. Notez le message d'avertissement. Vous pouvez désactiver ce certificat lorsque vous ne déboguez pas avec Charles.
    7. Maintenant, vous pourrez voir toutes les données envoyées et reçues dans Charles.

    Configuration de votre appareil Android

    Ces étapes ne fonctionneront que sur les appareils exécutant Android 2.3 et versions ultérieures.
    1. Sur votre appareil, ouvrez un navigateur et accédez à //chls.pro/ssl.
    2. Vous serez invité avec une boîte de dialogue Nom de certificat . Spécifiez un nom pour le certificat et définissez le champ Utilisé pour sur Wi-Fi. Sélectionnez OK pour installer le certificat.
    3. Ensuite, ouvrez le menu Paramètres > Wi-Fi . Vous verrez une liste des réseaux auxquels vous vous êtes connecté sur votre appareil. Sélectionnez le réseau sans fil auquel vous êtes actuellement connecté. N'oubliez pas que ce réseau doit correspondre au réseau auquel vous êtes connecté sur votre ordinateur.
    4. Cela ouvrira une petite boîte de dialogue avec trois options : Modifier, Afficher ou Oublier. Sélectionnez Modifier.
    5. Dans cette boîte de dialogue, faites défiler vers le bas et activez le champ Afficher les options avancées . Cela révélera les paramètres Proxy et IP.
    6. Modifiez le champ Paramètres du proxy sur Manuel.
    7. Remplissez le champ Nom d'hôte proxy avec l'adresse IP de votre ordinateur et le champ Port proxy pour correspondre au port défini dans Charles (par défaut, le port est défini sur 8888). Sélectionnez Enregistrer .
    8. Dans le navigateur par défaut de votre appareil, accédez à un site Web. Vous pourrez voir toutes les données envoyées et reçues dans Charles.
