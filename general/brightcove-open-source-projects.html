Projets Open Source Brightcove

Vous trouverez ci-dessous une liste de tous les projets Open Source actuels pour les technologies Brightcove, regroupés par catégorie.
Les projets Open Source ne sont pas officiellement pris en charge par Brightcove.

Bienvenue sur Brightcove Open Source. Vous trouverez ici un certain nombre de projets construits par des employés de Brightcove ou des membres de la communauté Brightcove pour vous aider à accélérer votre développement Brightcove.

Si vous êtes intéressé à contribuer à des projets, rendez-vous sur les sites de projet individuels sur Github. Si vous avez un projet que vous souhaitez lister ici, contactez-nous.

API actuelles

  1. Script de dépannage de demande d'API

    Dépannez les requêtes HTTP et les réponses vers/depuis les API Brightcove.

    https://github.com/BrightcoveOS/api-request-troubleshooting-script

  2. Uploader Brightcove

    Module frontend pour téléchargement direct en plusieurs parties sur s3 avec sauce secrète pour Brightcove ingest.

    https://github.com/BrightcoveOS/evaporate-brightcove

  3. Wrapper d'API de lecture Brightcove

    Ceci est un wrapper PHP pour l' API Brightcove Playback écrite par Theresa Newman.

    https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

  4. Wrapper PHP pour Brightcove Dynamic Inest

    Ce projet fournit un point de départ pour l'intégration du Brightcove API d'ingestion dynamique dans votre candidature. Il fournit des moyens simples d'ajouter et de mettre à jour des vidéos et des médias associés en utilisant soit l'ingestion basée sur l'extraction soit le téléchargement de fichiers source.

    https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

  5. Façade API

    Cette humble bibliothèque se veut une façade simple sur les API serveur de Brightcove. Comme la foule exige des fonctionnalités supplémentaires, elle sera ajoutée ici.

    https://github.com/nwbb/node-brightcove

  6. Client API de gestion du lecteur

    Ce module peut être utilisé pour accéder à l'API Brightcove Player Management. La documentation de l'API sous-jacente peut être trouvée ici.

    https://github.com/BrightcoveLearning/Player-Management-Client

  7. Exemples d'API Brightcove utilisant CSharp

    Ce projet montre comment les nouvelles API Brightcove peuvent être utilisées avec C #.

    Veuillez noter que cela fonctionnera à partir d'environnements de bureau (testés), mais ne fonctionnera probablement pas s'il est utilisé dans les pages ASP.NET en raison de CORS. Si vous rencontrez ce problème, je pense que la seule façon de résoudre ce problème est d'utiliser le proxy PHP fourni par Brightcove dans tous leurs API CMS pages d'exemple.

    Tout le code est fourni tel quel et purement à titre d'exemple. Utilisez-le à vos risques et périls. Aucun support n'est fourni pour elle de quelque manière que ce soit, forme ou forme.

    https://github.com/houseofmackee/BrightcoveCS

  8. API Brightcove Outils Python

    Ce référentiel contient des outils et des modules écrits en Python pour gérer les API liées à Brightcove et simplifier les tâches telles que l'itération sur la vidéothèque complète et l'exécution d'actions sur les vidéos.

    Version (s) testée (s) de Python : 3.8.3

    Tout le code est fourni tel quel et purement à titre d'exemple. Utilisez-le à vos risques et périls. Aucun support n'est fourni pour elle de quelque manière que ce soit, forme ou forme.

    https://github.com/houseofmackee/BrightcovePY

  9. Client API de gestion du lecteur

    Ce module peut être utilisé pour accéder à l'API Brightcove Player Management. La documentation de l'API sous-jacente peut être trouvée ici.

    https://github.com/BrightcoveLearning/Player-Management-Client

Intégrations système

  1. Connexion vidéo Brightcove pour SharePoint 2013

    Brightcove Video Connect pour SharePoint 2013 permet aux éditeurs d'ajouter, de parcourir et de gérer des vidéos et des sélections vidéo directement à partir de SharePoint. Son composant lecteur permet d'intégrer facilement les ressources Brightcove Video Cloud directement dans les pages SharePoint.

    http://go.brightcove.com/bc-video-connect-sharepoint2013

  2. Brightcove Video Connect pour Drupal

    Brightcove Video Cloud est une solution d'hébergement et de publication vidéo destinée à fournir des vidéos de qualité professionnelle aux audiences numériques sur n'importe quel appareil. Il offre des lecteurs hautement personnalisables, un streaming HD, une synchronisation avec des CDN tiers et YouTube. Avec Brightcove Video Connect for Drupal, vous pouvez ajouter et gérer efficacement du contenu vidéo sur vos sites Web alimentés par Drupal.

    https://www.drupal.org/project/brightcove

  3. Brightcove Video Connect pour WordPress

    Vous cherchez à gérer votre bibliothèque Brightcove Vidéo et Playlist nativement à partir de WordPress ? Avec ce plugin, développé par 10up.com, vous avez le pouvoir de gérer plusieurs comptes et vidéothèques, de télécharger des vidéos et de les ajouter à des playlists, de rendre des shortcodes avec vos vidéos et tout cela à partir de l'interface d'administration WordPress.

    https://wordpress.org/plugins/brightcove-video-connect/

  4. Connecteur SharePoint 2013

    Fournit l'intégration entre SharePoint 2013 et plusieurs comptes Video Cloud.

    https://github.com/brightcove/sharepoint-connector

  5. Brightcove Video Connect pour Adobe Experience Manager

    Brightcove Video Connect for Adobe Experience Manager (AEM) est le moyen le plus simple d'intégrer le flux de travail entre Brightcove Video Cloud et AEM. Grâce à Brightcove Video Connect, les utilisateurs d'AEM peuvent ajouter, parcourir et gérer des vidéos directement depuis l'interface utilisateur d'AEM.

    http://go.brightcove.com/bc-video-connect-aem

  6. Composante CQ5

    Le composant CQ5 intègre Brightcove

    Technologies : CQ5

    https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

  7. chariot élévateur

    Salesforce fournit une API en bloc pour faciliter la synchronisation massive des données d'un magasin de données vers salesforce. Ce projet fournit une gem pour utiliser l'API Salesforce Bulk dans une application ruby.

    Technologies : Salesforce, Ruby

    https://www.github.com/brightcove/salesforklift

  6. Diamant

    Diamond est un démon python qui recueille des mesures système et les publie sur Graphite. Il est capable de collecter des métriques CPU, mémoire, réseau, i/o, charge et disque. En outre, il dispose d'une API pour implémenter des collecteurs personnalisés pour collecter des mesures à partir de presque toutes les sources.

    Technologies : Graphite, Python

    https://github.com/python-diamond/Diamond
