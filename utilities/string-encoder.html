Encodeur de chaîne

Utilité pour encoder les chaînes URI 3 façons.

Ajoutez les noms de méthodes suggèrent, ceux-ci sont généralement utiles lorsque vous avez besoin d'inclure une chaîne dans une URL. Lequel est approprié dépend de la situation. Voici quelques exemples de la façon dont nous les trouvons utiles.

  • escape est la méthode la plus ancienne, et fait le moins ; nous n'avons pas eu l'occasion de l'utiliser dans nos applications
  • encodeURI est généralement adéquat pour les chaînes qui seront passées comme valeurs pour les paramètres d'URL
  • encodeURIComponent est un encodeur le plus complet, mais généralement vous n'en avez besoin que pour les chaînes qui seront utilisées dans le cadre d'un domaine/chemin d'URL, ou si vous avez besoin de passer une URL en tant que paramètre (nous le faisons fréquemment en passant des requêtes API aux proxy via AJAX)
