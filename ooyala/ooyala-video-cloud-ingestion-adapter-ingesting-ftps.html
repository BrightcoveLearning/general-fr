Ooyala-Video Cloud Ingestion Adaptateur : Ingestion avec FTPS

Vous pouvez continuer à utiliser FTP avec un manifeste XML pour ingérer vos vidéos dans Video Cloud. Cette rubrique fournit une vue d'ensemble du processus.
Remarque : ce sujet s'applique seul aux clients d'Ooyala qui ont migré vers Video Cloud.

Introduction

Vous pouvez utiliser FTPS (recommandé) ou FTP (obsolète) pour télécharger des vidéos, des miniatures, des sous-titres codés et des fichiers manifeste sur Video Cloud.

Note : Cette rubrique s'applique à OIS v2.5.

Vue d'ensemble de FTP Ingest

À propos de FTPS

FTP est un protocole réseau standard pour transférer des fichiers d'un ordinateur à un autre. Pour OIS v2.5, FTPS (FTP sur SSL) est requis. La plupart des systèmes d'exploitation prennent en charge FTP nativement, et vous pouvez choisir parmi de nombreuses applications clientes FTP. Vous pouvez automatiser les téléchargements FTP en écrivant des scripts, demander aux utilisateurs de charger en bloc des vidéos via une application Windows, Mac ou Linux, ou demander aux utilisateurs de charger des vidéos en bloc via un navigateur (par exemple, FireFTP).

Note : Vous pouvez utiliser n'importe quel client FTP pour télécharger des vidéos.

Stratégie de compte

Votre compte est limité à un maximum de 100 Go sur le serveur d'ingestion à tout moment. Si vous prévoyez de télécharger plus de contenu, vous pouvez demander une augmentation temporaire de votre client Success Manager ou du support Brightcove.

Ingestion avec un client FTPS

Vous pouvez télécharger une vidéo via FTP/FTPS avec ou sans manifeste.

Note : Tous les fichiers de téléchargement doivent être conformes aux exigences d'ingestion décrites dans les formats d'ingest pris en charge

Téléchargement de vos fichiers via FTPS (OIS v2.5)

Note : OIS v2.5 nécessite un client FTPS.
  1. Préparez le (s) fichier (s) que vous souhaitez télécharger.
  2. Lancez votre programme client FTPS.
  3. Connectez-vous au point de terminaison d'ingest à l'aide des informations d'identification (e-mail et mot de passe) de votre compte utilisateur uniquement chargé de Video Cloud.
  4. Téléchargez le (s) fichier (s) que vous souhaitez ingérer (vidéos, miniatures, fichiers de sous-titres codés, etc.). Si vous utilisez un fichier manifeste, veillez à télécharger tous les fichiers référencés dans le fichier manifeste.
  5. Ooyala suggère que vous téléchargez le fichier manifeste en dernier.

Les fichiers sont supprimés juste après leur traitement. Les fichiers soumis qui ne passent pas par l'ensemble du flux de travail d'ingestion (par exemple, en attendant le téléchargement du fichier manifeste) sont supprimés du serveur après 7 jours.

Note : Les caractères spéciaux (non ASCII) dans les noms de fichiers ne sont pas pris en charge lorsque des fichiers sont ingérés via FTP et que l'ingestion basée sur le manifeste est utilisée.
