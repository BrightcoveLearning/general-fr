Ooyala-Video Cloud Ingestion Adapter : Ingester avec Aspera

Vous pouvez utiliser le logiciel de transfert de fichiers Aspera pour télécharger des vidéos, des miniatures, des sous-titres codés et des fichiers manifeste dans Backlot.
Remarque : ce sujet s'applique seul aux clients d'Ooyala qui ont migré vers Video Cloud.

Vue d'ensemble de Aspera Ingest

À propos d'Aspera

Aspera fournit un client de transfert de fichiers hautes performances pour télécharger des vidéos sur Backlot. Aspera utilise le chiffrement lors du transfert des données.

Stratégie de compte

Votre compte est limité à un maximum de 100 Go sur le serveur d'ingestion à tout moment. Si vous prévoyez de télécharger plus de contenu, vous pouvez demander une augmentation temporaire auprès de votre gestionnaire de compte ou du support Brightcove.

Téléchargement du client Aspera

Aspera est disponible en tant que plug-in de navigateur gratuit ou comme client de bureau sous licence. Le plugin du navigateur Aspera sera proposé au téléchargement la première fois qu'un utilisateur se connecte au point de terminaison ingest et se connecte avec succès. Vous pouvez également télécharger le logiciel séparément.

  • Pour télécharger le plugin gratuit du navigateur, allez sur la page de téléchargement d'Aspera et téléchargez le plugin Aspera Connect.
  • Pour télécharger le client de bureau, accédez au client aspera.

Ingester avec le client Aspera

Vous pouvez télécharger une vidéo via le client Aspera avec ou sans manifeste.

Note : Tous les fichiers de téléchargement doivent être conformes aux exigences d'ingestion décrites dans les formats d'ingest pris en charge

Téléchargement de vos fichiers via le client Aspera (OIS v2.5)

  1. Préparez le (s) fichier (s) que vous souhaitez télécharger.
  2. Dans un navigateur, entrez le point de terminaison transfer.ooyala.com. Ce point de terminaison fonctionnera à partir de tous les emplacements.
  3. Connectez-vous au point de terminaison ingest à l'aide des informations d'identification (e-mail et mot de passe) de votre compte d'utilisateur Backlot.
  4. Si vous y êtes invité, ouvrez l'application Aspera. (Si aucune invite n'apparaît, l'application Aspera est déjà ouverte.)
  5. Téléchargez le (s) fichier (s) que vous souhaitez ingérer (vidéos, miniatures, fichiers de sous-titres codés, etc.). Si vous utilisez un fichier manifeste, veillez à télécharger tous les fichiers référencés dans le fichier manifeste.
  6. Ooyala suggère que vous téléchargez le fichier manifeste en dernier.

Les fichiers sont supprimés immédiatement après leur traitement. Les fichiers soumis qui ne passent pas par l'ensemble du flux de travail d'ingestion (par exemple, en attendant le téléchargement du fichier manifeste) sont supprimés du serveur après 7 jours.
