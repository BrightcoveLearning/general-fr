Ooyala-Video Cloud : Formats de fichier manifeste

Cette rubrique couvre les formats de fichier manifeste que vous pouvez utiliser pour ingérer des vidéos et des ressources.
Remarque : ce sujet s'applique seul aux clients d'Ooyala qui ont migré vers Video Cloud.

Introduction

Lors de l'ingestion de vidéos, vous pouvez utiliser des fichiers manifeste pour spécifier des métadonnées supplémentaires (y compris le titre de la vidéo, la description, les heures de vol, les étiquettes et les métadonnées personnalisées), ainsi que des fichiers de vignettes et de sous-titres codés. Les fichiers Manifest sont au format XML ou CSV. Pour plus de détails sur les manifestes, voir :

  • Fichiers Manifest CSV
  • Fichiers Manifest XML

Lorsque le traitement commence

Note : Cette rubrique s'applique à OIS v2.5.
Note : Video Cloud ne commence pas à traiter les vidéos tant que le fichier manifeste XML ou CSV n'est pas reçu. Par conséquent, il est important de choisir une stratégie de téléchargement par lots. Si vos vidéos doivent être disponibles rapidement (p. ex. distribution de nouvelles), vous devez créer de très petits lots ou même les regrouper individuellement (un fichier CSV ou XML par vidéo).
